# SmileOn (스마일온)

언제든지 온라인(On)으로 미소(Smile)를 위한 예약/상담이 가능한 지능형 병원 관리 시스템

## 프로젝트 개요

AI 챗봇과 CRM을 결합한 자연어처리 기반 치과 관리 시스템 `SmileOn` 을 개발하고, 이를 적용한 가상 치과 `BellaDent` 웹 페이지를 구현했습니다.

- 프로젝트명: SmileOn (스마일온)
- 목표: 자연어처리 기반 지능형 병원 관리 시스템 구축
- 적용 병원: 벨라덴치과 (BellaDent)
- 개발기간: 2025년 5월 7일 ~ 2025년 6월 25일
- 팀원
  - 고대견 (팀장): 백엔드, 프론트엔드, 기획, 디자인
  - 최명심: 디자인, 프론트엔드
  - 윤성훈: 백엔드, 프론트엔드
  - 김민기: 백엔드, 프론트엔드

## 적용 병원 페르소나

- 상호: 벨라덴치과 (BellaDent)
- 위치: 광주광역시 남구 봉선로 87
- 개원 연도: 2019년
- 규모: 중형 치과 (의사 3명, 실장, 위생사, 상담사, 수납 포함 8명)
- 진료 과목: 일반치과, 교정과, 임플란트, 심미치료
- 주요 환자층: 20~50대 성인, 가족, 교정환자

## 주요 기능

### 병원 홈페이지 (벨라덴치과 사례)
- 메인페이지
- 회원가입 / 로그인 / 회원정보 수정
- 진료 예약 페이지
- 실시간 채팅 상담 (Socket.IO + AI 챗봇 적용)
- 환자 대시보드
- 공지사항 / FAQ 리스트 확인
- 치료 후기 작성 (이미지 포함)

### CRM (내부 시스템)
- 환자 / 직원 등록 및 수정
- 진료 기록, 예약 내역, 상담 내역 관리
- 예약-결제 연동
- 공지사항 등록 (홈페이지용 / 사내용 구분)
- 대기현황 모니터링
- 실시간 채팅 상담 (Socket.IO 적용)
- SMS 발송
  - 광고
  - 병원 처음 방문 시 안내 문자
  - 예약 내역 안내 문자
- 치료 후기 관리
- AI 챗봇 설정

### AI & 자동화
- 자연어 처리 기반 챗봇 (예약 / 상담 / FAQ 응답)
- 예약 시간 추천

## 기술 스택

### 프론트엔드
- React
- TailwindCSS

### 백엔드
- Node.js
- Express
- Socket.IO (실시간 채팅)

### 인증 / 데이터베이스
- Firebase Authentication
- Firebase Firestore
- Firebase Storage
- Firebase Cloud Functions

### AI
- Gemini API (AI 챗봇 / 자연어처리)
- OpenAI API (홈페이지 이미지 생성)

### 외부 API / 서비스
- SSODAA (https://ssodaa.com/) : SMS 전송
- Google Maps API : 지도 표시

- **최명심:** 디자인, 프론트엔드
- **윤성훈:** 백엔드, 프론트엔드
- **김민기:** 백엔드, 프론트엔드

## 코드 컨벤션

참고: [Udacity Git Commit Message Style Guide](https://udacity.github.io/git-styleguide/)

- `type: Subject` 형태 (50자 이내, 영문 동사원형, 첫 글자 대문자)
- 본문 (72자 이내, 무엇/왜 작성)
- footer (이슈 트래커 ID)

주요 type

| type | 설명 |
| --- | --- |
| Feat | 기능 추가 |
| Fix | 버그 수정 |
| Docs | 문서 수정 |
| Refactor | 리팩토링 |
| Design | UI 변경 |
| Chore | 빌드/환경 |
| Test | 테스트 추가 |
| !HOTFIX | 긴급 수정 |
| !BREAKING CHANGE | 주요 API 변경 |
